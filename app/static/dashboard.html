Batch OMR Grading Guide

To perform a successful batch class grading, please follow these guidelines:

  • Create the OMR assessment first and configure the correct answer key.
  • Prepare all student submission sheets as images (JPEG, JPG, or PNG formats).
  • Compress all submission images into a single ZIP archive. Make sure files are placed in the root of the ZIP file (not nested inside folders).
  • The backend image scanner will automatically extract student sheets, identify students based on layout characteristics, and check bubble marks.
  • You will be directed to the Interactive Review Wizard to verify the detected student names, inspect any double-marked bubbles, and finalize the submissions.
